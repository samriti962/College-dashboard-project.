Course Details: Database Management Systems (CIT105)

Programme Information

Programme: B.Tech. (IT), 2nd Year

Semester: 4

Credits: 4

Teaching Hours: 45(L) + 30(P) = 75 hrs

Total Marks: 150 (CA: 90, ESE: 60)

Duration of ESE: 3 hours

Course Type: Core Course

Prerequisites: ESC103, LCIT103

Course Outcomes

  • Apply knowledge of database systems, SQL and NoSQL.
  • Formulate effective database designs with dependencies and recovery.
  • Utilize query processing and normalization for relational data.
  • Design physical and object-relational databases for performance.
  • Analyze emerging technologies like NoSQL, cloud DBs, graph DBs.
  • Implement comprehensive database solutions for real-world applications.

Units

Unit 1: Introduction to Database System (7 hrs)

  • DBMS vs File Systems
  • Data Models: Relational, Hierarchical, Network, Object-Oriented
  • Database Languages (DDL, DML, DCL)
  • DBMS Architecture, Keys, Constraints
  • ER & EER Diagrams, Design Issues

Unit 2: Relational Model (7 hrs)

  • Relations, Attributes, Tuples, Keys
  • Relational Algebra & Calculus
  • SQL: Data Types, Queries, Joins, Subqueries
  • Tables, Triggers, Views, Indexing
  • Embedded SQL, Security & Authorization

Unit 3: Relational Database Design (7 hrs)

  • Functional & Multivalued Dependencies
  • Normal Forms (1NF–5NF, BCNF)
  • Decomposition & Dependency Preservation

Unit 4: Transaction Management & Concurrency Control (7 hrs)

  • ACID properties
  • Failure & Recovery
  • Concurrency control, Serializability
  • Two-phase locking, Timestamp, Validation protocols
  • Deadlocks, Logs, Logging protocol

Unit 5: Recovery & Security (5 hrs)

  • Failure Classification
  • Log-based Recovery, Shadow Paging
  • Recovery with Concurrent Transactions
  • Checkpoints, Backup & Restore
  • Database Security, SQL Injection Prevention

Unit 6: NoSQL Database (6 hrs)

  • Introduction & Terminology
  • Document, Key-Value, MongoDB
  • Industry Use Cases: Metlife, Facebook, Google

Unit 7: Emerging Topics (6 hrs)

  • Real-time Databases, Streaming Data
  • Spatial & Multimedia Databases
  • Cloud-native DB Features: Scalability, Replication, Backup
  • AI-ready Databases: Vector DBs

Laboratory Work

  1. Creating & Managing Tables
  2. Basic SQL SELECT Statements
  3. Restricting & Sorting Data
  4. Manipulating Data (Insert, Update, Delete)
  5. Single Row Functions
  6. Displaying Data from Multiple Tables (Joins)
  7. Aggregating Data Using Group Functions
  8. Subqueries
  9. Creating Views
  10. Overview of MongoDB (CRUD operations)

Mini Project

Students must do a project from course contents in groups, submit a report, and give a presentation.

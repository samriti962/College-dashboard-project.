Course Details: Web Technologies (CIT106)

Programme Information

Programme: B.Tech. (IT), 2nd Year

Semester: 4

Credits: 4

Teaching Hours: 45(L) + 30(P) = 75 hrs

Total Marks: 150 (CA: 90, ESE: 60)

Duration of ESE: 3 hours

Course Type: Core Course

Prerequisites: ESC103, LCIT103

Course Outcomes

  • Design and structure web pages using HTML5.
  • Style and layout web content using CSS3 and Bootstrap.
  • Add interactivity to websites using JavaScript.
  • Build simple dynamic websites using PHP and MySQL.
  • Develop full web projects with form validation and CRUD operations.
  • Understand client-server interaction in basic web applications.

Units

Unit 1: Introduction to Web Designing (7 hrs)

  • Basics of Web Development
  • Website Structure: HTML, CSS, JavaScript
  • Client-side vs. Server-side Overview
  • Static vs. Dynamic Websites
  • Web Development Tools: VS Code, Browsers, Developer Tools

Unit 2: HTML5 and CSS3 – Structure of Web Pages (10 hrs)

  • HTML Document Structure
  • Headings, Paragraphs, Links, Images
  • Semantic Tags
  • Lists, Tables, Forms and Input Types
  • Embedding Multimedia: Video, Audio, iFrames
  • CSS Syntax, Selectors, Properties
  • Box Model: Margin, Border, Padding, Content

Unit 3: JavaScript – Web Interactivity (6 hrs)

  • JavaScript Basics: Variables, Operators, Control Structures
  • Functions and Events
  • DOM Manipulation
  • Form Validation

Unit 4: Responsive Design with Bootstrap (7 hrs)

  • Bootstrap Grid System
  • Components: Icons, Buttons, Navbar, Badges, Alerts, Toolbars, Panels
  • Pagination, Tables, Cards, Forms

Unit 5: Introduction to PHP (9 hrs)

  • Definition, Features, Server-side Basics
  • Installation (LAMP/XAMPP/WAMP)
  • PHP Syntax, Variables, Data Types, Constants
  • Output using echo and print
  • Control Structures
  • User-defined Functions & Built-in Functions
  • File Inclusion: include, require
  • Working with HTML Forms: $_GET, $_POST, Validation

Unit 6: PHP and MySQL – Server-side Programming (6 hrs)

  • Introduction to MySQL Database
  • Connecting to MySQL (mysqli, PDO)
  • Executing SQL Queries from PHP
  • CRUD Operations: Insert, Read, Update, Delete
  • Displaying Data in HTML Tables

Laboratory Work

  1. Design a basic static webpage using HTML5.
  2. Develop a styled webpage using CSS3.
  3. Create a responsive webpage using media queries.
  4. Design an interactive webpage using JavaScript.
  5. Build a multipage website using Bootstrap.
  6. Install and configure XAMPP/WAMP and write basic PHP scripts.
  7. Develop PHP scripts with control structures and functions.
  8. Create HTML forms and handle data with PHP.
  9. Design modular PHP applications using include/require.
  10. Build a dynamic PHP+MySQL CRUD application.

Mini Project

Students must do a project from course contents in groups, submit a report, and give a presentation.

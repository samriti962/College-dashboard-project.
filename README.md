# Academia Dashboard (Student & Teacher Portal)

This project is a simple HTML and CSS based dashboard system for students and teachers.  
It provides an interface where users can access academic information like courses, attendance, assignments, exams, results, timetable and other academic services.

## Features

- Student Dashboard
- Teacher Dashboard
- Attendance Calendar
- Course List
- Assignments Section
- Examination Information
- Result Page
- Timetable Page
- Holiday Information
- Salary Page (Teacher)

## Technologies Used

- HTML
- Internal CSS

## Project Structure

index.html – Main dashboard page 
ps.html – Profile Student
pt.html – Profile Teacher
assignment.html – Assignment section  
attendance.html – Attendance calendar  
courses.html – Course list  
event.html – Event information  
exam.html – Examination details  
fee.html – Fee information  
holiday.html – Holiday list  
result.html – Result page  
salary.html – Teacher salary page  

## Purpose of the Project

This project was created for learning and practicing basic web development concepts using HTML and CSS.

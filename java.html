Course Details: Object Oriented Programming with JAVA (CIT107)

Programme Information

Programme: B.Tech. (IT), 2nd Year

Semester: 4

Credits: 5

Teaching Hours: 45(L) + 60(P) = 105 hrs

Total Marks: 150 (CA: 90, ESE: 60)

Duration of ESE: 3 hours

Course Type: Core Course

Prerequisites: ESC103

Course Outcomes

  • Recognize principles of OOP for robust programs.
  • Apply logic and syntax to computational problems.
  • Analyze and evaluate software designs.
  • Implement advanced Java features for applications.
  • Utilize JDBC for persistent data management.
  • Develop, design, debug programs using Java OOP.

Units

Unit 1: Introduction to Java (4 hrs)

  • History, Evolution, Byte Code
  • Buzzwords, OOP Paradigms
  • Abstraction, Principles of OOP
  • Structure of Java Program, Environment

Unit 2: Data Types, Operators & Control Statements (7 hrs)

  • Data Types, Literals, Variables, Type Casting
  • Arrays, Operators (Arithmetic, Bitwise, Relational, Logical)
  • Conditional Operator, Precedence
  • Selection & Control Statements
  • Recursion vs Iteration

Unit 3: Classes, Objects & Methods (7 hrs)

  • Classes, Objects, Reference Variables
  • Methods, Constructors, Overloading
  • Objects as Parameters, Returning Objects
  • ‘This’ Keyword, Garbage Collection, finalize()
  • Access Specifiers, Static, Final, Command Line Arguments

Unit 4: Inheritance (5 hrs)

  • Basics, Super, Method Overriding
  • Dynamic Dispatch, Abstract Classes
  • Final with Inheritance, Derived Class Constructors
  • Object Class

Unit 5: Packages & Interfaces (4 hrs)

  • Packages, Access Protection
  • Importing Packages
  • Defining & Implementing Interfaces
  • Nested & Extended Interfaces
  • Default Interface Methods

Unit 6: Exception Handling (5 hrs)

  • Model, Types, Try-Catch, Multiple Catch
  • Nested Try, Throw, Finally
  • Built-in & Custom Exceptions
  • Chained Exceptions

Unit 7: Multithreaded Programming (5 hrs)

  • Thread Lifecycle, Main Thread
  • Creating Threads, Multiple Threads
  • isAlive(), join(), Priorities
  • Synchronization, Inter-thread Communication
  • Suspending, Resuming, Stopping Threads

Unit 8: String Handling (3 hrs)

  • Constructors, Length, Operations
  • Character Extraction, Comparison, Searching
  • Modification, Conversion, Case Change
  • StringBuffer

Unit 9: Java Database Connectivity (5 hrs)

  • JDBC Architecture
  • Drivers (Type 1–4)
  • DriverManager, Connection, Statement
  • PreparedStatement, CallableStatement, ResultSet
  • Connection Establishment, CRUD Operations

Laboratory Work

Includes 29 experiments covering JDK setup, data types, operators, control statements, recursion vs iteration, classes & objects, constructors, inheritance, packages, interfaces, exception handling, multithreading, string handling, and JDBC CRUD operations.

Mini Project

Students must prepare a project in groups using Java concepts (applets, multithreading, JDBC). Submit report (8–10 pages) and give presentation.

Course Details: Advanced and Emerging Trends in IT (CIT108)

Programme Information

Programme: B.Tech. (IT), 2nd Year

Semester: 4

Credits: 4

Teaching Hours: 45(L) + 30(P) = 75 hrs

Total Marks: 150 (CA: 90, ESE: 60)

Duration of ESE: 3 hours

Course Type: Core Course (Integrated)

Prerequisites: BSIT101, CIT102, ESC103, ESC105

Course Outcomes

  • Apply fundamental AI and ML principles to develop intelligent solutions.
  • Analyse and interpret complex datasets for decision-making.
  • Perform reconnaissance and vulnerability analysis using NMAP, Wireshark, Burp Suite.
  • Describe full-stack web and mobile development environments (MEAN, MERN, Flutter).
  • Assess emerging trends like Web3, DAOs, Layer-2, decentralized AI.
  • Design and develop basic web applications integrating APIs.

Units

Unit 1: Introduction to Artificial Intelligence (6 hrs)

  • Definition, scope, history, applications, branches of AI
  • Types of intelligent agents and environments
  • Problem solving approaches, search strategies
  • Ethical and societal aspects of AI
  • Knowledge representation, logic, inference rules
  • Bayesian networks, fuzzy logic

Unit 2: Machine Learning Fundamentals (8 hrs)

  • Supervised, Unsupervised, Reinforcement learning
  • ML workflow, preprocessing, feature extraction
  • Linear Regression, Decision Trees
  • Clustering (K-Means), Neural Networks basics
  • Python tools: Scikit-learn, TensorFlow basics

Unit 3: Data Science and Big Data Analytics (9 hrs)

  • Data Science lifecycle, applications
  • Data collection, cleaning, normalization, encoding
  • Exploratory Data Analysis (EDA): histograms, box plots, correlation
  • Big Data concepts: 4Vs, challenges
  • Python libraries: Pandas, NumPy, Matplotlib
  • Big Data ecosystem: Hadoop, HDFS, Spark

Unit 4: Network Security and Ethical Hacking (6 hrs)

  • Network security models
  • Information gathering tools: Google search, whois, dig, NMAP
  • Packet sniffing: Wireshark, Burp Suite
  • Password attacks, social engineering, DoS attacks

Unit 5: Full Stack Web & Mobile Development (8 hrs)

  • MEAN, MERN stacks
  • Front-end frameworks overview
  • Back-end and API concepts
  • Mobile development: Flutter, Android Studio basics
  • Project structure, widgets, UI design

Unit 6: Blockchain and Decentralized Technologies (8 hrs)

  • Evolution from Bitcoin to Web3
  • Centralized vs. Decentralized systems
  • Structure of blockchain: blocks, hashes, chains
  • Types: Public, Private, Consortium, Hybrid
  • Applications: DeFi, NFTs, CBDCs, Supply Chain, Healthcare
  • Future trends: Layer-2, DAOs, Decentralized AI

Laboratory Work

  1. Data Handling and Preprocessing using Python
  2. Exploratory Data Analysis and Visualization
  3. Implementation of a Simple ML Model
  4. Network Sniffing using Wireshark
  5. Phishing demo: Social networking login page
  6. Basic MERN/MEAN Demo App
  7. Basic Flutter/Android Studio Demo App
  8. Exploring Blockchain Structure using Python
  9. Simulating Cryptocurrency Transaction (MetaMask, Solidity)
  10. Blockchain Use Case Demo – Supply Chain Tracking

Mini Project

Students must do a project from course contents in groups, submit a report, and give a presentation.

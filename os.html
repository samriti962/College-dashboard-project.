Course Details: Operating Systems (CIT104)

Programme Information

Programme: B.Tech. (IT), 2nd Year

Semester: 4

Credits: 4

Teaching Hours: 45(L) + 30(P) = 75 hrs

Total Marks: 150 (CA: 90, ESE: 60)

Duration of ESE: 3 hours

Course Type: Core Course

Prerequisites: BSC102, ESC105

Course Outcomes

  • Exemplify types of Operating Systems, deadlocks, process, file & memory management.
  • Implement deadlock scheduling algorithms.
  • Analyze memory & file management mechanisms.
  • Classify page replacement algorithms for demand paging.
  • Use disk scheduling algorithms for external memory utilization.
  • Examine case studies of OS (Windows, UNIX, Linux, Cloud, Virtualization).

Units

Unit 1: Introduction (6 hrs)

  • Types of OS: Batch, Multiprogrammed, Time-sharing, Real-time, Distributed, Parallel
  • Kernel & Shell functions
  • General structure, OS services, System calls

Unit 2: Process Scheduling & Management (8 hrs)

  • Processes & Threads, States, PCB
  • Schedulers: Pre-emptive & Non-preemptive
  • Scheduling Algorithms
  • Inter-process Communication
  • Synchronization: Critical sections, Mutual Exclusion, Semaphores

Unit 3: Memory Management (9 hrs)

  • Logical vs Physical address space
  • Fragmentation, Partitioned memory
  • Paging, Segmentation, Segmentation with paging
  • Virtual memory, Demand Paging
  • Page Replacement Algorithms: FIFO, Optimal, LRU
  • Thrashing, Local vs Global replacement

Unit 4: Deadlocks (5 hrs)

  • Conditions for deadlock
  • Resource allocation graphs
  • Prevention, Avoidance, Detection, Recovery

Unit 5: Secondary Storage & File Management (10 hrs)

  • Disk structure, Disk scheduling (FCFS, SSTF, SCAN, C-SCAN, LOOK, CLOOK)
  • Disk management, Formatting, Boot blocks, Bad blocks, SSD management
  • File concepts: Types, Access methods, Attributes, Operations
  • Allocation methods: Contiguous, Linked, Indexed
  • File System Architecture, Distributed File Systems
  • Advanced Protection Mechanisms

Unit 6: Case Studies (7 hrs)

  • Windows, UNIX, Linux
  • OS support for Cloud, Virtualization, Containerization

Laboratory Work

Includes 14 experiments covering OS installation, virtualization, CPU scheduling, producer-consumer problem, Banker’s algorithm, memory allocation, page replacement, disk scheduling, file allocation, Linux commands, and shell programming.

Mini Project

Students must do a project from course contents in groups, submit a report, and give a presentation.
